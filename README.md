## ⬇ Установка

1. Скачайте **`MovementHubNe-Setup-x.x.x.exe`** из [Releases](https://github.com/nelaff/movementhub-ne/releases/latest)
2. Установите (Windows 10/11)
3. Запустите **movementhub.ne** → войдите или зарегистрируйтесь
4. Активируйте ключ из [@neluff_gie_bot](https://t.me/neluff_gie_bot)

> Без лицензии приложение не откроется. Ключ приходит сразу после оплаты в боте.
